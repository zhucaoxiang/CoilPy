Brief instruction to use coilpy.py¶

Author: Caoxiang Zhu (czhu@pppl.gov)

This is a very brief instruction showing how to use my personal python script coilpy.py. This is based on Python 2.7. A new version based on Python3 is under developing.

There are about 100 functions in the source file. Please check each function for more details.
